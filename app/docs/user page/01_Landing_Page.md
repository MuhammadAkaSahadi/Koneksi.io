# HALAMAN 1: LANDING/BERANDA

## Layout Desktop (>1024px)

### Navigation Bar (Fixed, Height: 72px)
*   Background: White dengan shadow halus (0 2px 8px rgba(17, 100, 184, 0.08))
*   Container: Max-width 1280px, padding horizontal 48px
*   Layout: Logo kiri | Menu tengah (Katalog, Tentang, FAQ) | CTA kanan (Masuk + Daftar Gratis)
*   Logo: "Koneksi.io" dengan icon abstrak menghubungkan dua node (Tech Blue)
*   CTA Daftar: Background Volt Green, rounded-lg (8px), hover effect glow

### Hero Section (Height: 600px)
*   Layout: 2-column grid (60% konten, 40% visual)
*   Background: Gradient halus dari white ke #F8F9FA
*   Konten Kiri:
    *   Badge kecil: "Platform Pembelajaran Teknologi Mandiri" (Tech Blue, rounded-full)
    *   H1: "Bangun Kompetensi Tech dari Nol hingga Siap Industri"
    *   Subtitle: Penjelasan singkat tentang fokus software & IoT
    *   Dual CTA:
        *   Primer: "Mulai Belajar Gratis" (Volt Green, large, rounded-lg)
        *   Sekunder: "Lihat Katalog" (outline Tech Blue)
    *   Trust indicator: "2 sub-bab gratis setiap tema • Tanpa kartu kredit"
*   Visual Kanan:
    *   Ilustrasi isometric workspace dengan elemen IoT board, kode, dan device
    *   Warna dominan Tech Blue dengan aksen Volt Green

### Value Proposition Section (Padding: 96px vertical)
*   Background: White
*   Layout: 3-column grid (gap 32px)
*   Setiap Card:
    *   Icon: Circle 64px (background Volt Green opacity 10%, icon Tech Blue)
    *   Title H3: Fitur unggulan (misal: "Self-Paced Learning", "Project-Based", "Freemium Model")
    *   Description: 2-3 kalimat
    *   Card style: Border 1px #E5E7EB, rounded-xl (12px), padding 32px
    *   Hover: Border Tech Blue, subtle lift (translateY -4px), shadow enhance

### Featured Courses Preview (Padding: 96px vertical)
*   Background: #F8F9FA
*   Section Header:
    *   H2: "Mulai Perjalanan Tech-mu"
    *   Subtitle: "Pilih jalur Software atau Hardware (IoT)"
*   Layout: 2-column grid untuk kategori besar

*   Card tema: (lihat detail di Halaman Katalog)

### CTA Final Section (Padding: 96px vertical)
*   Background: Tech Blue gradient
*   Container: Centered, max-width 800px
*   Content:
    *   H2 putih: "Siap Memulai Pembelajaran?"
    *   Subtitle putih opacity 90%
    *   Large CTA: "Daftar Sekarang — Gratis" (Volt Green button)

### Footer (Padding: 64px vertical)
*   Background: #1F2937 (dark)
*   Layout: 4-column grid
    *   Column 1: Logo & tagline
    *   Column 2: Link produk (Katalog, Tentang, FAQ)
